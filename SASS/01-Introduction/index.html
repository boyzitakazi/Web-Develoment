Introduction

SASS merupakan salah satu css prerocessor kita dapat menggunakan logika, nasting, mixin, dan masih banyak lagi, SASS biasa digunakan ketika projeknya itu sudah cukup besar kemungkinan menggunakan SASS karena program nantinya dapat lebih mudah dibaca

SASS vs SCSS

ada dua pilihan ketika kita menggunakan SASS yaitu pada ekstensi programnya, contoh .sass program tidak lagi menggunakan tutup kurung kurawal, sedangkan .scss menggunakan kurung kurawal sama seperti css biasa namun bedanya program dapat melakukan nasting, membuat variable, layaknya bahasa pemrograman pada umumnya.
